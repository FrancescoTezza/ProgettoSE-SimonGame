ProgettoSE-SimonGame, progetto Android sviluppato in Kotlin con Jetpack Compose.

Informazioni sull'ambiente di sviluppo:
- Sistema operativo: macOS
- IDE utilizzato: Android Studio Panda 3.3
- Linguaggio: Kotlin
- Emulatore utilizzato : Medium Phone API 36.1

NOTA SUL CODICE :
In fondo al file 'MainActivity.kt' è stata commentata una parte di codice considerevole che presenta delle classi '@Preview'.
Ciò è dovuto al fatto che, a causa delle limitazioni hardware del mio dispositivo, l'applicazione Android Studio non mi permettesse di scaricare nessun tipo di emulatore.
Le '@Preview' sono state quindi utili durante lo sviluppo dell'applicazione per controllare la disposizione degli elementi sia in modalità portrait che landscape.
Prima della consegna ho fatto affidamento ad un mio collega di corso, che ha eseguito il codice con l'emulatore indicato sopra, facendo si che mi potessi accertare del corretto funzionamento logico nel rispetto dei requisiti richiesti.
